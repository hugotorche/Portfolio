Python libraries: Pandas | Datetime | Os | Re I decided to analyze YouTube data about Roland Garros editions from 2014 to 2020. My goal was to define which edition was the best according to the data and get my hands on YouTube API. The analysis was performed with the help of:
  • Corey Schafer tutorial about YouTube API
  • Playlist and Video data from YouTube API
Observations and Conclusion:
  • Roland Garros 2019 edition is the one with most highlights
  • Final and Semi-Final highlights are the most popular and this popularity has increased over time
